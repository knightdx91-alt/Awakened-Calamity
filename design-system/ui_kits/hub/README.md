# UI Kit — Hub (launcher)

The dark-cyber launcher / landing. A centered cyan-gradient wordmark, lowercase
section labels (Play / Create / World), and a stack of `LaunchCard`s with
geometric `MapMarker` icons. Clicking a card prints a System-flavored boot line.

**Files**
- `index.html` — composes `LaunchCard` + `MapMarker` from the design-system bundle.

This is the bridge surface between the warm game and the cold System: dark cyber
panels, cyan double-strokes, Courier-family body. Use it as the template for any
menu/landing/launcher outside the game world.
