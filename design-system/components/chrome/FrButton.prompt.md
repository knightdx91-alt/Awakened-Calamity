FireRed action / toggle button — inverts to solid red on hover or when `active`. Use for settings toggles, option groups, and battle actions.

```jsx
<FrButton active>Pause-on-act</FrButton>
<FrButton>Live gauge</FrButton>
<FrButton size="sm" disabled>Locked</FrButton>
```

In a group, set `active` on exactly one. `size="sm"` for dense rows.
