The world-map place glyph — the brand's core iconography. Shape encodes place type; an optional hazard halo rings it.

```jsx
<MapMarker type="safe" label="Dawnhearth" />
<MapMarker type="calamity" hazard="cold" label="Frostfoam" />
<MapMarker type="dungeon" />
```

Shapes: diamond=Safe, rotated square=Holdfast, circle=route, triangle=dungeon, star=Calamity, dashed circle+?=Hidden, rings=water/underwater. Reuse these shapes anywhere you denote a place or danger class.
