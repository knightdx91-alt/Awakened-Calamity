Contents
========
RPG Maker VX Ace Run Time Package (RTP), version 1.00, extracted from the
official installer (Setup.exe / Setup-1.bin). Folders:
  Audio/    — BGM, BGS, ME, SE (Ogg Vorbis)
  Graphics/ — Animations, Battlebacks, Battlers, Characters, Faces, Parallaxes,
              Pictures, System, Tilesets, Titles, etc. (PNG)
  Fonts/    — VL Gothic / VL PGothic (the RTP's bundled fonts; see their own
              license files under Fonts/VLGothic/)
  ReadMe.txt — the original RTP readme + terms as shipped by the publisher.

Source
======
Downloaded from the Internet Archive item `rpgvxace-rtp_202606`
(RPGVXAce_RTP.zip), then unpacked with `innoextract` (no Windows installer run).
This is the same RTP the publisher distributes for free.
