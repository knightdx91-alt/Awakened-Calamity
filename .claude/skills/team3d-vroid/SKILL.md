## VRM (Virtual Reality Model) Format

### VRM Overview
- **Format**: Open format for 3D avatars in VR/AR applications
- **Compatible**: Compatible with various VTuber applications
- **Standard**: Based on glTF 2.0 standard
- **Extensions**: VRM-specific extensions for avatar features
- **Metadata**: Includes avatar metadata and licensing
- **Cross-Platform**: Works across multiple platforms

### VRM Features
- **Blendshapes**: Facial expressions and lip sync
- **Bone Structure**: Standardized bone structure for animation
- **Physics**: Physics for hair, clothing, and accessories
- **Materials**: PBR materials for realistic rendering
- **Metadata**: Avatar metadata and licensing information
- **First Person**: First-person view settings

### VRM Compatibility
- **VSeeFace**: Popular VTuber software
- **Luppet**: VTuber software for macOS
- **VMagicMirror**: VTuber software for Windows
- **VRChat**: Social VR platform
- **VRoid Hub**: Platform for sharing VRM avatars
- **Custom Applications**: Custom VTuber applications
