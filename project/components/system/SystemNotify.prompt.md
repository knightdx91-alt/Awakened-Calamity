The System's notification toast — the antagonist's voice, in cold neon. Use for every moment the System acts on the player: offers, stat changes, audits, lockouts.

```jsx
<SystemNotify type="info" message="Stamina increased to 64." />
<SystemNotify type="warning" message="Unauthorized Safe Zone detected. Register for protection? (Cr + Surveillance)" />
<SystemNotify type="danger" message="AUDIT INBOUND. A Construct has joined the enemy side." />
```

Write copy in menacing corporate cheer — polite, bureaucratic, faintly threatening. Keep the `[ THE SYSTEM ]` label unless you have a reason not to.
