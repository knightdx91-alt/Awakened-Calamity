How to use

• Pick a Tileset tab, then a tile, and paint.
• 🌱 Auto / ▦ Tiles toggle (Ground): Auto blends terrain edges; Tiles = raw — drag a box in the palette for a multi-tile stamp.
• Layer: paint Ground then Overlay (buildings/props). Other layer dims.
• Tools: Pencil / Rect / Ellipse / Fill / Pick / Eraser.
• Map List (bottom-left): press-and-hold or right-click for New / Rename / Delete.
• Export writes <layout>.json + <map>.json.
