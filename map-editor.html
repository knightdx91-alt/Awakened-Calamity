How to use

• A · Auto = autotile terrains (edges blend as you paint).
• B · Tiles = raw tiles. Drag a box in the palette to grab a multi-tile stamp.
• Layer: paint Ground then Overlay (buildings/props). Other layer dims.
• Tools: Pencil / Rect / Ellipse / Fill / Pick / Eraser.
• Export writes <layout>.json + <map>.json.
